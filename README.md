## 📱 Android Programming

Repository này được tạo để lưu trữ các bài thực hành, bài tập và project trong môn **Lập trình Android**.  
Mục tiêu của repository là giúp sinh viên làm quen với việc phát triển ứng dụng trên hệ điều hành Android bằng **Java và Android Studio**.

Nội dung bao gồm:
- Các bài thực hành trên lớp
- Các bài tập về Activity, Layout, và xử lý sự kiện
- Làm việc với dữ liệu (SQLite / Firebase)
- Project cuối kỳ phát triển ứng dụng Android

Repository cũng được sử dụng để theo dõi quá trình học tập và phát triển kỹ năng lập trình Android.
